# Youtubeのエンゲージメント予測

## 概要
YouTubeは現代において最も影響力のある情報共有プラットフォームの一つであるが、動画投稿者が「視聴者からどう評価されるか（高評価・低評価）」を予測することは極めて困難である。

本課題では、Kaggleの「Youtube_Trending_Dataset」を活用し、動画のタイトル、カテゴリ、タグ、投稿時間などのメタデータから、その動画の「エンゲージメント（Like/Dislike数）」を予測するモデルを構築する。

本研究の目的は、単に数値を予測することに加え、「どのような動画特性が高評価を集め、どのような特性が低評価につながるのか」という隠れたルールをデータマイニングによって明らかにすることにある。

## 実験の進め方
### *Phase1: とりあえず線形回帰モデルを作ろう。*
1. データセットファイルを読み込む
2. 場合によってはデータセットを加工する
3. 特徴量を絞る(一番始めのモデルだから、簡単に視聴回数とかトレンドに載った日とかの数値だけでやる)
4. 線形回帰のモデルを準備
5. データセットをテスト用と学習ように分別
6. 構築したモデルに対し、テスト用の入力データを与えた際の結果を出力

### *Phase2: データを加工して別のモデルを作ってみよう*
1. データセットの文字列型のカラムを加工し、新しい特徴量を作る

    例: titleからタイトル文字数に変換とか
2. 線形回帰モデルに組みこむ

### *Phase3: 線形回帰モデル以外のモデルを作ろう*
多項式回帰モデルとか、ランダムフォレストとかを使ってみる。
