# tag_count

YouTubeトレンド動画のタグの個数を動画ごとにカウントするファイル群。
`english_titles.csv` を入力データとして使用する。

## ファイル一覧

### スクリプト

| ファイル名 | 説明 |
|---|---|
| `tag_count.py` | `english_titles.csv` から動画ごとのタグ個数を集計し `tag_count.csv` を出力する。タグは`\|`区切りでカウントし、`[none]`またはタグなしの場合は0とする。タグ数の平均・中央値・最大・最小もコンソールに表示する。 |

### 出力ファイル

| ファイル名 | 説明 |
|---|---|
| `tag_count.csv` | 動画ごとの video_id・タイトル・タグ個数の一覧（タグ数 降順） |

## 実行方法

```
python tag_count/tag_count.py
```

`info-dm-g5/` ディレクトリから実行する。
